Here's how to quickly install and setup Minimal Mistakes Plus for use with
GitHub Actions.

** Using Minimal Mistakes Plus Scaffold (The Fastest Approach)

1. Clone or download the repository [[https://github.com/josephtesfaye/minimal-mistakes-plus-scaffold][Minimal Mistakes Plus Scaffold]].
2. In its top folder run:

   #+begin_src sh
   bundle install
   bundle exec jekyll serve
   #+end_src
